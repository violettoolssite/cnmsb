核心功能

让命令行更高效

01

智能内联建议

输入命令时，光标后自动显示灰色预测文字。不用按任何键触发，边输入边看建议。按 Tab 或右箭头接受。

02

交互式选择器

按 Tab 打开补全选择菜单，显示所有可选项及说明。上下方向键切换选项，再按 Tab 确认。Esc 取消。

03

问号帮助模式

在命令后输入 ? 查看所有可用选项。按类别分组显示：常用选项、文件选项、输出选项等，快速了解命令用法。

04

组合参数补全

支持类似 tar -zxvf 的组合参数。输入 tar -z 后会建议继续添加 x、v、f 等常用组合参数，无需逐个输入。

05

智能路径补全

自动补全文件和目录路径。目录用亮青色显示，普通文件用绿色，隐藏文件用灰色，一眼区分类型。

06

历史命令模式

按 Alt+H 打开专门的历史命令选择器。只显示匹配当前输入前缀的历史命令，实时筛选，快速重用。

07

实时更新

输入或删除字符时，补全列表实时更新，无需重新按 Tab。响应迅速，体验流畅，像 IDE 一样丝滑。

08

模糊匹配

不需要精确输入，支持模糊匹配。输入 gti 也能匹配 git，输入 dkr 也能匹配 docker，容错性强。

09

内置编辑器

自带 cntmd 文本编辑器，类 vim 操作。输入时自动显示补全建议，预装 100+ 常用词汇，实时学习。

Vim 风格

熟悉的操作方式

如果你用过 vim，上手 cntmd 毫无障碍。支持 Normal、Insert、Command 三种模式， h/j/k/l 移动光标，:w 保存，:q 退出。

  • i/a/o 进入插入模式
  • Esc 返回普通模式
  • :w 保存 :q 退出 :wq 保存退出
  • :q! 强制退出不保存

  • .py 文件 → #!/usr/bin/env python3
  • .js 文件 → // JavaScript
  • .rs 文件 → // Rust

安装

几行命令，开始使用

Debian / Ubuntu

# 下载 deb 包
wget https://github.com/violettoolssite/cnmsb/releases/download/v0.1.0/cnmsb_0.1.0_amd64.deb

# 安装
sudo dpkg -i cnmsb_0.1.0_amd64.deb

# 重新登录后生效
# 包含: cnmsb, cntmd 编辑器

源码编译

# 克隆仓库
git clone https://github.com/violettoolssite/cnmsb.git
cd cnmsb/cnmsb-tool

# 编译 (需要 Rust 1.70+)
cargo build --release

# 安装
sudo ./install.sh

安装后会自动配置 Zsh 环境，将用户默认 shell 改为 zsh
