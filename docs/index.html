8 大核心功能

类似 IDE 的补全体验，让命令行操作更高效

01

智能内联建议

输入命令时自动在光标后显示灰色预测文字，无需按任何键触发。按 Tab 或右箭头接受建议。

02

交互式选择器

按 Tab 打开补全选择菜单，使用上下方向键切换选项，再按 Tab 确认选择。按 Esc 取消。

03

问号帮助模式

在命令后输入 ? 查看该命令的所有可用选项，按类别分组显示，快速了解命令用法。

04

组合参数补全

支持类似 tar -zxvf 的组合参数补全。输入 tar -z 后会建议继续添加 x、v、f 等参数。

05

智能路径补全

自动补全文件和目录路径。目录使用亮青色显示，普通文件使用绿色，一眼区分。

06

历史命令模式

按 Alt+H 打开专门的历史命令选择器，只显示匹配当前输入的历史命令，实时筛选。

07

实时更新

输入或删除字符时，补全列表实时更新，无需重新按 Tab 触发。响应迅速，体验流畅。

08

颜色区分

亮黄色显示选项和命令，亮青色显示目录，绿色显示文件和子命令，灰色显示描述。

安装 cnmsb

选择适合你的安装方式

# 下载 deb 包
wget https://github.com/violettoolssite/cnmsb/releases/download/v0.1.0/cnmsb_0.1.0_amd64.deb

# 安装（会自动配置 zsh 环境）
sudo dpkg -i cnmsb_0.1.0_amd64.deb

# 重新登录后生效
# 克隆仓库
git clone https://github.com/violettoolssite/cnmsb.git
cd cnmsb/cnmsb-tool

# 编译（需要 Rust 1.70+）
cargo build --release

# 安装
sudo ./install.sh
# 在 ~/.zshrc 中添加以下内容
source /usr/share/cnmsb/cnmsb.zsh

# 重新加载配置
source ~/.zshrc

# 使用 deb 包安装会自动完成上述配置
